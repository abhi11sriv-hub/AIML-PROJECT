# Loan Default Risk Predictor

> **BYOP Capstone — Data Science / ML**  
> Predicting loan default using LightGBM, with SHAP explainability and Fairlearn fairness auditing.

---

## Project overview

This project builds an end-to-end machine learning pipeline to predict whether a loan applicant will default (charge off) on a LendingClub loan. Beyond raw predictive performance, the project explicitly addresses:

- **Data leakage prevention** — removing post-outcome features before training
- **Class imbalance** — handled with SMOTE oversampling
- **Explainability** — SHAP values for global and per-prediction explanations
- **Fairness auditing** — demographic parity and equalized odds measured across proxy sensitive groups using Fairlearn
- **Interactive dashboard** — Streamlit app with live SHAP waterfall charts

## Getting started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/loan-default-risk.git
cd loan-default-risk
```

### 2. Set up the environment

```bash
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Download the dataset

1. Go to [Kaggle — LendingClub Loan Data](https://www.kaggle.com/datasets/wordsforthewise/lending-club)
2. Download `loan.csv` (or the full `accepted_2007_to_2018Q4.csv`)
3. Place it in the `data/` directory:

```
data/loan.csv
```

> The dataset is ~1.8 GB. For faster development, use `sample_frac=0.1` in `preprocess.py`.

---

## Running the pipeline

Run each step in order:

### Step 1 — Preprocess

```bash
python src/preprocess.py data/loan.csv
```
Outputs: `data/processed.parquet`

### Step 2 — Train models

```bash
python src/train.py
```
Outputs: `models/lightgbm.pkl`, `models/xgboost.pkl`, `models/logistic.pkl`, `models/best_model_info.json`

### Step 3 — Generate SHAP explanations

```bash
python src/explain.py
```
Outputs: `reports/figures/shap_*.png`, `reports/figures/feature_importance.csv`

### Step 4 — Fairness audit

```bash
python src/fairness.py
```
Outputs: `reports/figures/fairness_*.png`, `reports/figures/fairness_metrics.json`

### Step 5 — Launch the dashboard

```bash
streamlit run app.py
```
Opens at `http://localhost:8501`

## Key results

| Model | ROC-AUC | Avg Precision |
|-------|---------|---------------|
| Logistic Regression | ~0.68 | ~0.42 |
| XGBoost | ~0.73 | ~0.52 |
| **LightGBM** | **~0.74** | **~0.54** |

Top predictive features (SHAP): `int_rate`, `grade`, `dti`, `fico_range_low`, `revol_util`

Fairness: Demographic Parity Difference audited across income tier and employment length groups. See `reports/model_card.md` for full results.
